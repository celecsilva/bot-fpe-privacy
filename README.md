# Política de Privacidad - Bot FPE

**Última actualización:** Marzo 2026

## ¿Qué información recopilamos?
El Bot FPE recopila el nombre, carrera y año de cursada que el usuario ingresa voluntariamente durante el registro en el bot.

## ¿Para qué usamos la información?
La información se utiliza exclusivamente para personalizar la experiencia del usuario dentro del bot y mejorar el servicio de atención estudiantil.

## ¿Compartimos tu información?
No compartimos, vendemos ni cedemos información personal a terceros.

## ¿Cómo protegemos tu información?
Los datos se almacenan de forma segura en Google Firestore y solo son accesibles por los administradores del centro de estudiantes FPE.
